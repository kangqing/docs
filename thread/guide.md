# MacOS idea 快捷键指南

## 查找
```bash
# 查找类
command + o

# try ... catch 等快捷键
# 先写被 try 的一行，然后
option + command + t

# 自动补全
option + enter

# 删除选中行
commmand + del

# 另起一行
shift + enter

# 查看最近操作的文件
command + E

# 进入光标的方法/变量的接口或定义处
command + B

# 行注释
command + /

# 块注释
command + option + /

# 复制当前行
command + D

# 快捷创建get set 方法
command + N

# 弹出首选项，系统配置
command + ,

# 格式化代码
command + option + L

# 重构代码成一个方法
command + option + M 

# 大小写切换
command + shift + U

# 关闭当前页
command + W

# 跳转到实现类，可以忽略接口
command + option + B


```
